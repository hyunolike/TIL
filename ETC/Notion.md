## Notion 단축키🤗

#### 많이 쓰는 단축키
- `Ctrl` + `N` : 새로운 페이지 생성
- `Ctrl` + `Shift` + `N`: 새로운 Notion 창 생성
- `Ctrl` + `P`: 최근에 본 페이지 열기, 검색, 이동
- `Ctrl` + `[`: 기존 페이지로 돌아감
- `Ctrl` + `]`: 페이지 앞으로 이동
- `Ctrl` + `Shift` + `L`: 다크모드 변경
- `Ctrl` + `\`: 왼쪽 bar 없애기
