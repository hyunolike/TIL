## 자바스크립트란 무엇인가?🤔
> 프로그래밍 언어 이다!!

단어에 __Java__ 가 들어가 있지만 현재는 Java와 관련이 없다.<br>
`스크립트 언어`와 `인터프리터 언어`

자바스크립트가 널리 사용되어지는 이유도 `웹브라우저` 가 있기에 가능
> 동적으로 컨텐츠를 바꾸고 UI를 이동시키는 등 여러가지 일을 수행한다.

또한 크롬브라우저에서 사용하는 자바스크립트 엔진을 이용해서 서버사이드 애플리케이션을 만들 수 있는 `Node.js` 가 등장했다.

---
#### 객체(Object)🤬
```
함수, 클래스(틀) => 객체, 개체, object
```

1. __생성자 함수__ 로 객체 만들기
```
function 틀(){} => new 틀()
```
- 잘 생각해보면 `function` 키워드가 생성자 함수이다.
    > 표준내장객체라고 한다. `new Function()`

2. __Object__ 로 객체 만들기(권장되는 방법이 아니다)
```
new Object()
```

`😁프로토타입 체인` (.prototype) : 모든 객체 연결해서 사용 가능하다.
> 프로토타입을 이용한 __상속받기__

3. __객체 리터럴__
```JavaScript
const a = {};
```
잠깐만!!!😱 화살표 함수는 `this`를 받지 않아~~ 상황에 맞게 사용하자!

#### 표준 내장 객체🙃
대표적으로 `Object()` 외에 많이 있다.
> MDN 표준내장객체 확인
